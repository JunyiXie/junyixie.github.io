About

Hello! My name is Junyi Xie, and I’m a software engineer with experience at ByteDance, Alibaba.

I have expertise in Virtual Machine, Compiler, JIT, parallel optimization, binary size optimization, and hotfixing for C++/Swift.

My projects include:

  • designing and implementing a method-based JIT (ARM64) for CPython
  • optimizing CPython runtime support for parallel execution (subinterpreters)
  • optimizing CPython binary size, designing and implementing a hookable indirect function dispatch for the LLVM compiler backend
  • designing and implementing ByteDance crash reporter system, designing and implementing iOS crash reporter, designing and implementing iOS & Android crash symbol & aggregation system
  • optimize the performance of Taobao’s app and optimizing development environments.
